**The Problem:**
So you love Project Quarm and are a big fan of Zeal and QuarmTool, but you are always lagging behind on version since there is no main patcher for the game. This is where Quarm+ comes into play.

**The Solution:**
Quarm+ asks for your main Project Quarm and QuarmTool directories and saves them to a local settings.json file. Otherwise, it is a simple patcher that not only loads the newest versions of Zeal, QuarmTool, and the Quarm Patcher into your selected directories, but also:

- Allows for installation of functional older versions through the version selector
- Offers Nillipus UI for quick installation in /uifiles/ folder, as well as a custom UI importer via Git or EQInterface, for highest level of enjoyment
- Easily removes incompatible UI's from TAKP2.2 installation
- Offers a quick and easy "Apply Suggested Settings" button which applies the most valuable optimizations to eqclient.ini, as well as a more robust "Advanced Configurator" which allows for easy customization of the most valuable ini settings
- Offers a quick modification button to change movement keys to WASD, as well as a few other hotkey changes that make immediate gameplay easier
- Offers a quick installation of KaichoFX spell effects with backup of SpellEffects folder for rolling back changes if needed
- Quick access buttons to open QuarmTool, Quarm Patcher, PQ Companion and Project Quarm directly from the patcher window
- Compatible with all Quarm-adjacent and TAKP servers

Whether you are in the Discord and up to speed, or you're making the devs scream because you're 10 versions behind, Quarm+ is a tool that can help save you time, worry, and frees you from the lossy streams of #zeal-discussion and #open-source-discussion.

Feel free to share suggestions, and thank you for using Quarm+!

**How to run:**
- Download and run the Quarm+ exe from any directory
- Point to the Zeal directory (main TAKP/EQ directory) and QuarmTool directory
- Read, click, and REJOICE!
- Repeat this process about once per week (aka: whenever stuff starts breaking) or integrate it into your Project Quarm launch process
- KABOOM! You've just been Quarm+'d
